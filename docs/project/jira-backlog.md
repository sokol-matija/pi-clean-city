## Notes

This backlog represents the complete feature set for the CleanCity municipal problem reporting platform. The system has three main user types:

1. **Citizens (Građani)** - Report problems, track status, communicate with authorities
2. **Municipal Workers (Građanski Radnici)** - Process reports, update status, communicate with citizens
3. **Administrators** - Manage system, view statistics, generate reports

Key feature areas include:

- Report submission and tracking
- Interactive map visualization
- User management and authentication
- Comments and communication
- Admin dashboard and statistics
- Notifications system
- Mobile responsiveness
